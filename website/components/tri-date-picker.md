# tri-date-picker

## Props

<!-- @vuese:tri-date-picker:props:start -->
|Name|Description|Type|Required|Default|
|---|---|---|---|---|
|type|选择的类型|`'date'` / `'daterange'` / `'datemulti'` / `'year'` / `'month'`|`false`|'date'|
